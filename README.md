# AinaQuest (Unity)

Digital version of the AinaQuest card game built in Unity.

## Features (Current)
- Dynamic player count (1–5 players)
- Card drafting system (pick → pass)
- NPC opponent (solo mode)
- Card reveal system
- Player/NPC field tracking
- Running score system

## Tech Stack
- Unity 6 (C#)
- ScriptableObjects for card data
- Custom UI system (Canvas + Layout Groups)

## Next Steps
- Multiplayer support
- Card synergy / bonus system
- Improved UI/UX
- Animations and polish
